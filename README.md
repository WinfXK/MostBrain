# MostBrain
一个没卵用的小游戏-PVP And PVE

**MostBrain是一个多人PVP小游戏！并且附带部分PVE功能**

***他包含多种附加技能，每种技能都有自己不一样的BUff***


##它当前满足以下条件：
> - [x] 完全的文本自定义
- [x] 可免拆包更改显示的文本（全局）
- [x] 包含多个正面，负面的Buff
- [x] 可支持已存在或未来将会存在的任何经济核心（续用户自行添加）
- [x] 可自行添加用户自定义Buff
- [x] 支持设置游戏区域是否封闭

> 游戏时长短游戏不尽兴？完全的自定义游戏时间！可自定义道具是否发送补给！还不够？那就来自定义Buff吧！
